# Dog Class in Python

This project is a simple example of Object-Oriented Programming (OOP) in Python.
It creates a `Dog` class with basic attributes and a method to display a message.

## Features

* Implements a Python class using OOP principles
* Defines and initializes instance attributes (`name` and `breed`)
* Includes a class method to demonstrate behavior (`bark()`)

## Example Output

Woof! My name is Buddy and I'm a Golden Retriever.
